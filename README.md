# Customer-Churn-Prediction-Retention-Analytics-System
https://drive.google.com/drive/folders/122ym61Blotm6Lg7OgOx5-m4kj-RbcvnB?usp=sharing

Developed an end-to-end machine learning solution to predict customer churn and identify high-risk customers for retention campaigns.

Performed data preprocessing, feature engineering, missing value handling, and exploratory data analysis (EDA) to uncover churn patterns.

Built and compared multiple classification models including XGBoost, LightGBM, and CatBoost.

Evaluated model performance using ROC-AUC, Precision, Recall, F1-Score, and Confusion Matrix metrics.

Ranked customers based on churn probability to support targeted retention strategies.

Applied feature importance analysis and SHAP explainability techniques to identify key churn drivers and improve model transparency.

Generated business-focused insights using Lift Charts, Gain Charts, and Top-Decile Analysis to measure campaign effectiveness and potential ROI.

Completed as the final capstone assignment for the Matsuo Laboratory Machine Learning Program, demonstrating the application of advanced machine learning techniques to a real-world business problem.
